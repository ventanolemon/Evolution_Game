# 🧬 Эволюция

Головоломка на основе 2048: соединяй одинаковых существ, чтобы они эволюционировали.
Проведи жизнь от Амёбы до Киборга!

---

## Быстрый старт

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск

```bash
python main.py
```

---

## Как играть

### Цель

Соединяй две одинаковые плитки — они сливаются и превращаются в следующую ступень эволюции.
Достигни **Киборга** (10-я стадия) — и ты победил.

### Цепочка эволюции

```
Амёба → Рыба → Лягушка → Ящерица → Птица → Заяц → Волк → Медведь → Человек → Киборг
```

### Управление

| Клавиша | Действие |
|---|---|
| ← ↑ → ↓ | Сдвинуть все плитки в выбранную сторону |
| **R** | Начать заново (не выходя в меню) |
| **M** | Включить / выключить звук |
| **ESC** | Вернуться в главное меню |

При каждом ходе все плитки на поле скользят до упора в выбранную сторону.
Если в конце движения встречаются две одинаковые плитки — они сливаются,
а очки добавляются к счёту. После хода на свободной клетке появляется
новая плитка (Амёба или Рыба).

### Очки

За каждое слияние начисляются очки: чем выше стадия эволюции, тем больше очков.
Итоговый счёт сохраняется в таблице лидеров — туда можно попасть через главное меню.

### Конец игры

- **Победа** — на поле появился Киборг.
- **Нет ходов** — поле заполнено, ни одно слияние невозможно.
- **Время вышло** — таймер добрался до нуля (только в режимах с таймером).

## Режимы игры

| Режим | Таймер | Особенность |
|---|---|---|
| ⏱ **Классика** | 90 с | +1.5 с за каждое слияние (до 3 за ход) |
| ⚡ **На время** | 45 с | Без бонусов — только скорость |
| ♾ **Бесконечный** | нет | Играй пока есть ходы |

## Карты

Перед игрой можно выбрать карту — каждая меняет расположение поля и добавляет
особые клетки, которые усложняют или помогают:

| Карта | Размер | Описание |
|---|---|---|
| 🟫 **Обычное поле** | 4×4 | Классика без сюрпризов |
| 🌿 **Сад эволюции** | 5×5 | Апгрейдеры помогают быстро продвинуться |
| 🌋 **Огненная тропа** | 4×4 | Огонь уничтожает плитки — будь осторожен |
| 🧱 **Лабиринт** | 5×5 | Стены делят поле на изолированные секции |
| 🚪 **Коридор** | 3×6 | Узкое поле — каждый ход на счету |
| 💀 **Испытание** | 6×6 | Стены, огонь, апгрейдеры и деградаторы сразу |

### Специальные клетки

| Клетка | Эффект |
|---|---|
| 🧱 **Стена** | Непроходима. Разрывает линию сдвига: плитки по разные стороны стены движутся независимо |
| 🔥 **Огонь** | Плитка, остановившаяся на этой клетке, уничтожается. Одноразовая |
| ⬆ **Апгрейдер** | Плитка на клетке получает +1 стадию эволюции. Одноразовый |
| ⬇ **Деградатор** | Плитка теряет 1 стадию. Если плитка была Амёбой — она исчезает. Одноразовый |

Специальные клетки срабатывают только один раз: после применения эффекта клетка
становится обычной пустой.

## Достижения

В игре есть 9 достижений, которые разблокируются автоматически по ходу игры.
Прогресс сохраняется между сеансами.

| Достижение | Условие |
|---|---|
| 🔗 Первое слияние | Слить две плитки в первый раз |
| ⭐ Полтысячи | Набрать 500 очков за одну партию |
| 🌟 Две тысячи! | Набрать 2000 очков за одну партию |
| 🐺 Вожак стаи | Получить плитку «Волк» (7-я стадия) |
| 🧑 Homo sapiens | Получить плитку «Человек» (9-я стадия) |
| 🤖 Киборг создан! | Победить — эволюционировать до Киборга |
| 🎮 Постоянный игрок | Сыграть 5 партий |
| ⚡ Гонщик | Победить в режиме «На время» |
| ♾ Без остановки | Набрать 2500 очков в бесконечном режиме |

При получении нового достижения в правом верхнем углу экрана появится уведомление.

## Таблица лидеров

После каждой партии игра предлагает ввести имя и сохранить результат.
Таблицу лидеров можно открыть из главного меню — результаты хранятся
отдельно для каждого режима.
