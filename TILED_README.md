# Создание карт в Tiled

## Установка

Скачай и поставь Tiled с [mapeditor.org](https://www.mapeditor.org/) (бесплатно, все ОС).

## Быстрый старт: открой существующую карту

1. Запусти Tiled.
2. **Файл → Открыть** → `EvolutionGame-fixed/maps/labyrinth.tmx`.
3. Tiled автоматически подхватит тайлсет `tilesets/special_cells.tsx`.
4. Смотри, редактируй, сохраняй.

## Создание новой карты с нуля

### Шаг 1. Новая карта

**Файл → Новый → Новая карта**

| Параметр | Значение |
|---|---|
| Ориентация | Ортогональная |
| **Формат слоя тайлов** | **CSV** (важно!) |
| Порядок отрисовки | Справа снизу |
| Размер карты | Фиксированный, 4×4 — 10×10 |
| Размер тайла | 64 × 64 |

Сохрани сразу как `maps/moe_pole.tmx` в папке проекта.

### Шаг 2. Подключи тайлсет

**Карта → Добавить внешний тайлсет** → выбери `maps/tilesets/special_cells.tsx`.

В панели "Тайлсеты" появятся 4 иконки: стена, огонь, апгрейдер, деградатор.

### Шаг 3. Свойства карты

**Карта → Свойства карты** → добавь **String**-свойства:

| Имя | Значение (пример) |
|---|---|
| `id` | `moe_pole` |
| `name` | `Моё поле` |
| `description` | `Проверяю механику огня` |

Игра возьмёт `name` и `description` для отображения в меню выбора карт.

### Шаг 4. Слой тайлов

1. В панели "Слои" убедись, что есть один слой тайлов.
2. **Переименуй его в `cells`** (ровно так, иначе парсер не найдёт).
3. Если слоя нет — правой кнопкой → Новый слой тайлов.

### Шаг 5. Рисуй

Кликай по тайлу в панели "Тайлсеты", затем клетки по полю. Пусто — просто ничего не ставь.

### Шаг 6. Сохрани

**Файл → Сохранить**. Файл должен быть `.tmx`.

## Семантика клеток

| Тайл | Эффект в игре |
|---|---|
| 🧱 Стена | Непроходимая. **Разрывает линию сдвига на секции** — плитки слева/справа от стены движутся независимо |
| 🔥 Огонь | Плитка, **остановившаяся на клетке**, уничтожается. Клетка гаснет (одноразовая) |
| ⬆ Апгрейдер | Плитка +1 стадия эволюции. Клетка гаснет |
| ⬇ Деградатор | Плитка −1 стадия. На стадии 0 плитка уничтожается. Клетка гаснет |

## Проверка карты

Запусти игру → Играть → выбор режима → **карта появится в списке автоматически**.

Если карта не появляется — посмотри в консоль: парсер печатает `⚠ ...` с объяснением причины.

## Частые ошибки

| Ошибка в консоли | Что делать |
|---|---|
| `не найден слой 'cells'` | Переименуй слой в точно `cells` (без пробелов, без заглавных) |
| `поддерживается только CSV` | Правка → Настройки → Формат слоя тайлов = CSV |
| `rows=... вне (2,10)` | Размер карты должен быть от 2×2 до 10×10 |
| `неизвестный GID` | Не тот тайлсет. Удали старый тайлсет и подключи `special_cells.tsx` |

## Добавление собственных типов клеток

1. Добавь PNG в `assets/cells/` (64×64, pixel-art стиль).
2. Открой `special_cells.tsx` в Tiled → выбери тайл → добавь custom-property **`type`** со значением (например `ice`).
3. В `src/game/cell_types.py` добавь значение в enum `CellType.ICE = "ice"`.
4. В `src/game/board.py::_apply_cell_effects` добавь ветку обработки.
5. В `src/game/cell_types.py::CELL_SPRITES` добавь путь к PNG.

Ядро игры не требует других изменений.
